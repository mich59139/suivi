MODE D’EMPLOI — Version “Responsables”

But : d’abord attribuer des domaines à des responsables, puis suivre les actions.

1) Sur le site (index.html)
- Accueil → Par outil / Par responsable / Par thème
- Cliquer une carte → liste d’actions → déplier

2) Dans Admin (admin.html)
- Onglet “Responsables” : attribuer des domaines (outils / thèmes)
- Bouton “Appliquer” : remplit automatiquement le champ “responsable” des actions non attribuées

3) Publication GitHub
- Cette version-ci est volontairement “simple” (pas de token).
- Pour publier en direct dans GitHub, utilise la version Admin GitHub (celle d’avant).
